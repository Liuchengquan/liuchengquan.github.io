python更新

2017-05-08

###python更新###
因为系统自带的python版本是2.7 所有我想要给更新成最新的版本，试过了好多中方法但是总会遇到各种各样的错误。但是总有一种方法可以成功，所以来写下来，希望能给大家带来方便。

  1. 因为我是使用的brew安装，没有安装的朋友可以安装一个试试，非常的方便，当然安装python的方法有很多种，也可以去官网下载
    brew install python3 安装python3
  2. 删除Mac自带的Python2.7 (这一步骤是非必须的，删除以后xcode打不开，因为xcode 运行是要跑基于python2.7的脚本)
    sudo rm -R /System/Library/Frameworks/Python.framework/Versions/2.7
  3. 把新下载的Python 移动到系统位置
    sudo mv /Library/Frameworks/Python.framework/Versions/3.5 /System/Library/Frameworks/Python.framework/Versions
    tips:在上面的过程中可能会出现权限的问题，是因为OS X EI Capitan 使用了Rootless保护系统内核,默认的情况下会锁定/system /sbin /usr 这三个目录，所以说我们如果要操作这几个目录的时候就需要先关闭这个保护。方法也很简单。重启电脑进入恢复模式Command-R，实用工具-终端 输入 csrutil disable 关闭内核保护，csrutil enable 恢复保护。
  4. 修改文件所属Group权限
    sudo chown -R root:wheel /System/Library/Frameworks/Python.framework/Versions/3.5

  5. 删除系统自带的Python link 并更改指定link

  6. 重新链接可执行文件

查看Python版本：python -V 结束
