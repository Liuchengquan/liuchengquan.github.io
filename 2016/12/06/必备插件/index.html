Homebrew 是什么？

Homebrew是一款Mac平台下的软件包管理工具，拥有安装，卸载，更新，搜索 等功能。只需要简单的命令就可以实现包的管理，而不用关心各种依赖和文件路径等问题。

#使用Homebrew查找，安装和卸载软件包
brew search git
brew install git
brew uninstall git
